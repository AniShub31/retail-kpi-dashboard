# Retail KPI Dashboard: Sales, Inventory & Performance Analytics

This project analyzes retail sales data to provide key performance indicators (KPIs) and insightful visualizations. Built with Python and data visualization libraries, it helps track sales trends, customer demographics, and product category performance.

---

## Project Structure

retail-kpi-dashboard/
├─ README.md # Project overview and instructions
├─ data/
│ └─ retail_sales_dataset.csv # Retail sales dataset for analysis
├─ analysis/
│ └─ retail_kpi_analysis.ipynb # Jupyter notebook with KPIs and visualizations
├─ images/
│ └─ *.png # Exported charts and plots

---

## Features

- Data cleaning and processing with pandas
- Calculation of key retail KPIs:
  - Total Sales
  - Average Order Value (AOV)
  - Quantity Sold
  - Sales by Product Category, Gender, and Age Group
- Monthly sales trend analysis
- Data visualizations using matplotlib and seaborn
- Exported charts saved in `/images/`

---

## Dataset

The dataset used is the [Retail Sales Dataset from Kaggle](https://www.kaggle.com/datasets/mohammadtalib786/retail-sales-dataset).  
Key columns include: `Transaction ID`, `Date`, `Customer ID`, `Gender`, `Age`, `Product Category`, `Quantity`, `Price per Unit`, and `Total Amount`.

---

## How to Run

1. Download the dataset CSV into the `/data/` folder.
2. Open the `retail_kpi_analysis.ipynb` notebook in Jupyter or Google Colab.
3. Run all cells to perform the analysis and generate charts.
4. View the exported visualizations inside `/images/`.

---

## Sample Insights

- Monthly sales show consistent growth with holiday season peaks.
- Accessories and Footwear are top-performing categories.
- Customers aged 18-30 contribute the highest sales.
- Female customers have higher purchase volumes during promotions.
- Average order size is about 2.5 items per transaction.
